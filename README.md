## 🚀 **Quick Start**

<div align="center">

### Get up and running in 3 simple steps!

</div>

<table>
<tr>
<td align="center" width="33%">

**1️⃣ Clone**
```bash
git clone https://github.com/yourusername/nimbusnet.git
cd nimbusnet
```

</td>
<td align="center" width="33%">

**2️⃣ Install**
```bash
npm install
# or
yarn install
```

</td>
<td align="center" width="33%">

**3️⃣ Launch**
```bash
npm run dev
# or
yarn dev
```

</td>
</tr>
</table>
